# データ構造

## 倫理的配慮

本研究で収集したデータは、研究倫理の観点から公開していません。以下にデータ構造を記載しますので、同様の研究を実施する際の参考としてください。

## 尺度の説明

### 自己効力感（Q1-Q4）
- 1: 全く自信がない
- 2: あまり自信がない
- 3: どちらでもない
- 4: 少し自信がある
- 5: とても自信がある

### 態度・有用感（Q5-Q16）
- 1: 全くそう思わない
- 2: あまりそう思わない
- 3: どちらでもない
- 4: ややそう思う
- 5: とてもそう思う

## データの前処理

1. **重複回答の処理**: 同一匿名コードで複数回答がある場合、タイムスタンプが最新のものを採用
2. **対応付け**: 事前・事後を匿名コードでマージ（inner join）
3. **欠測処理**: 項目ごとに欠測を除外して分析
